Here I have a collection of project I have been working on that have interested me. They include my Aspen simulations of various active pharmaceutical ingredients, MatLAB scripts to simulate non-ideal mixtures, and Processing scripts that utilize mathematical functions to create interesting patterns and visualize chemical engineering concepts. The animation folder contains the videos I have created utilizing processing. The Aspirin Plant and Ibuprofen plant contain files related to the modelling of those processes via Aspen Plus. NRTL Modelling contains the MatLab script for analyzing an example mixture and conducting a McCabe Thiele analysis on its distillation. Finally Processing Code contains the code I made to generate those animations. There are separate README in each folder to get an idea of how each project works.
